# Portfólio – Helton

## Estrutura
- index.html (Sobre mim)
- formacao.html (Formação)
- portfolio.html (Portfólio)
- contato.html (Contato)
- styles.css
- script.js
- assets/ (opcional)

## Como executar localmente
1. Abra a pasta no VS Code.
2. Dê duplo clique em `index.html` para abrir no navegador.

## Publicar no GitHub Pages
1. Crie um repositório público no GitHub com nome: `helton-portfolio` (ou similar).
2. Faça commit e push de todos os arquivos para a branch `main`.
3. No GitHub, vá em Settings > Pages.
4. Em “Source”, selecione `Deploy from a branch`.
5. Selecione `Branch: main` e `Folder: /root`.
6. Salve. Aguarde alguns minutos.
7. O site ficará acessível em: `https://SEU-USUARIO.github.io/helton-portfolio/`.

## Dicas
- Substitua links de projetos por seus repositórios e páginas reais.
- Ajuste textos, cursos e certificações para seu histórico.
- Mantenha seu nome no endereço do site conforme a exigência da atividade.
